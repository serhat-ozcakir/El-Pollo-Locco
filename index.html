El Pollo Loco

How to Play
Section 1: Kontrols
  • ⬅️ ➡️ Move
  • ⬆️ Jump
  • D key -> Throw Bottle
Section 2: Game Objective
  • Collect coins and bottles / bubbles.
  • Defeat enemies and the end boss.
  • Keep an eye on your health bar; if it empties, the game is over./li>
Section 3: Tips
  • Some enemies can only be defeated by jumping on top of them.
  • Use bottles / bubbles to damage enemies and the end boss.
  • Your character will Idle or Sleep when not performing actions.
Section 4: Extra Features
  • You can mute/unmute the game sounds using the mute button.
  • On mobile, the game can only be played in landscape mode; a warning is shown if held vertically.
